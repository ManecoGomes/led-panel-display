# LED Panel Display System

Sistema de exibição para painéis LED de 382x382 pixels.

## Páginas

- `/` - Painel de Profissionais
- `/imoveis` - Painel de Imóveis

## Recursos

- Scraping automático do site WordPress
- Rotação automática de 6 segundos
- Ordem aleatória dos slides
- Badges VENDE/ALUGA automáticos

## Tecnologias

- React + TypeScript
- Express.js
- PostgreSQL
- Tailwind CSS
